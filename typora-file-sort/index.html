Typora侧边栏文件列表排序-按修改时间

2020-03-31

Typora侧边栏文件视图默认排序方式是自然排序,即按照数字或者首字母顺序排序.

但是我自己用起来很不方便,比如我新建一个文件,重命名之后就会被排序到列表中间某个位置,不太好找.

所以想要按照修改时间倒序,这样新建的文件就在最上面了.

一开始右键菜单里找了半天没找到,后来才发现在列表的最下面.

非常简单如图操作,切换升序降序只需要再多点击一次图标就行:
