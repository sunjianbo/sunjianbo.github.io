Horde、SquirrelMail和RoundCube

2014-09-24

GoDaddy主机的cPanel集成了3中web邮件客户端，分别是Horde、SquirrelMail和RoundCube，于是进行了一番简单了解。

以下内容引用自

http://www.cnbeta.com/articles/179498.htm

SquirrelMail：

SquirrelMail是最流行的基于PHP的免费的，开源的Webmail. SquirrelMail有着众多强大的插件以及特性。在共享主机环境下，SquirrelMail平台变现不错。SquirrelMail很容易进行配 置和安装，SquirrelMail已经内置了PHP支持以及IMAP和SMTP协议。但是，SquirrelMail的GUI图形化界面表现得不是很棒。

下面是SquirrelMail的一些特性：

● 拼写检查

● 邮件过滤器系统（不知道有没有Gmail的过滤器强大）

● 基于网页的管理系统

● 日历以及地址簿

● 支持文件夹管理

RoundCube：

Linux下五个流行的Webmail

RoundCube 也是一个免费开源的Webmail, RoundCube有着难以置信的界面，并且很容易安装。RoundCube使用PHP和JavaScript语言编写。RoundCube支持多种语 言， 同时，RoundCube的图形化界面是兼容XHTML和CSS2的。

RoundCube部分特性：

● 提供私密保护功能

● 多个发件人支持

● 基于LDAP协议（轻量级目录访问协议，Lightweight Directory Access Protocol）的地址簿功能

● 超过70种语言的支持

● 插件扩展

● 邮件和联系人搜索

● 发送带有附件的邮件

● 文件夹管理和访问控制列表

● 不限制用户数量和用户数量

Horde：

首先，Horde是一个基于网页的Groupware.Horde基于PHP框架提供基本的邮件管理。 Horde提供IMP( internet massaging program)功能的邮件管理系统。日复一日，这个项目在开发人员和贡献者努力下提高了很多。Horde也是可以非常方便的安装和配置。在管理邮件、地 址簿和笔记方面，Horde有很多特性。Horde最近的稳定版本是 4.3.11. 在管理面板的设置方面有四个部分：常规选项、撰写、信息、其它。用户可以方便的自定义设置邮件的收发。

一些Horde特性：

● 文件和文件夹管理

● 快速设置收件箱缓存

● 垃圾邮件过滤

● 邮件过滤器

● 警告日志

● PGP支持

● 虚拟收件箱

● 笔记管理Mneno(Note manager)

● 任务管理 Nag(Task list manager)
