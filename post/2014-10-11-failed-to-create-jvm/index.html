打开eclipse报错：Failed to create the Java Virtual Machine

2014-10-11

昨天打开eclipse报错：

Failed to create the Java Virtual Machine

这个问题是常遇的，印象中是修改eclipse.ini文件，网上搜了一下，果然，都说是将 --launcher.XXMaxPermSize 后面的256m修改为128m。

照做一下发现不好使，瞬间崩溃。

后来又发现还有一种解决方案，就是在openFile下一行添加

-vm和JRE下的javaw.exe地址。

经试验成功打开。

总结如下：

遇到Failed to create the Java Virtual Machine有两种解决方案，一种不好可采用另一种，都是修改eclipse文件目录下的eclise.ini文件

  1. 将 --launcher.XXMaxPermSize 后面的256m修改为128m。

  2. 在openFile下一行添加-vm和JRE下的javaw.exe地址。
