修改图片宽度自适应使文字图片分开并一目了然

2014-09-25

发现发布的文章图片和文字混杂在一起，尤其是图片边缘是白色的话更是分不清哪里是文字哪里是图片了，这样关键内容不突出，体验非常不好。

于是想要修改样式，达到图片相应缩进的效果。

步骤如下：

1、打开当前主题目录下的style.css文件，ctrl+F找到img：如下

img {
	border:none;
	max-width:90%;
	height:auto;
	width:auto;
}

2、修改max-width值。这个是固定图片的最大宽度不超过所设置的值，这个值可以根据己设置的页面大小来调整。

可以输入像素单位如 800px，也可以像我这样直接输入百分比，这样浏览器放大缩小时也不会影响页面分布。

PS：

据说在代码块中增加如下代码可以针对在IE下失效的情况爆发小宇宙，如有需要，姑且一试。

width: expression(this.width > 780 ?"780px": (this.width+"px"));
